# Temporal.cheatsheet

A single-page reference for the [Temporal API](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Temporal) — modern date & time for JavaScript. Covers `Now`, `Instant`, `ZonedDateTime`, the `Plain*` types, `Duration`, conversions, calendars, and the shared API surface, with copy-to-clipboard snippets.

**Live:** https://emrebal98.github.io/temporal-cheatsheet/

## Run locally

```sh
python3 -m http.server 8000
```

Open http://localhost:8000.
